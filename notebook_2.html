σὺ δέ

The verbatim claim: “σὺ δ(έ): occurs predominantly in Homer and Herodotus,”

Our interpretation: If we searched for these two words next to each other, averaging authors' use of the phenomenon across all their works, Homer and Herodotus would be in the lead.

Requirements

To run and tweak this analysis, you will need the following:

  • this notebook, with its home directory

  • the pandas library

Averages

First, let’s import some necessary libraries. This second one is our bespoke query library for the Greek lit corpus.

Wow! That’s a lot of results. Let’s do some data wrangling to get a feel for how many of these results each author here is responsible for. Document URNs are individual identifiers for documents in the Greek lit corpus. The first four numbers correspond with an author, and the last four numbers correspond with a specific work. URNs will help us count these by author.

results['author'] = results['Document URN'].str.partition("-")[0]
real_results = results[results['Lemma'] == 'δέ']

authors = results['author'].unique()
hits = []

for a in authors:
    hit = len(real_results[real_results['author'] == a])
    hits.append(hit)

author_series = pd.Series(authors)
hits_series = pd.Series(hits)
author_data = pd.DataFrame({'author' : author_series, 'instances' : hits_series})

That’s more like it. Let’s see if we can use that list of unique authors to get a list of every URN they’re responsible for.

author_urns = []

for u in all_urns:
    for a in authors:
        if a in u:
            author_urns.append(u)

We want to see what percent of each author’s works use our phrase, so we will borrow some code from another notebook to get token and punctuation counts to help us get a word count for our goal.
