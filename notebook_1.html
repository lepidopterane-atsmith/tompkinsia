The Bow Stratagem

The verbatim claim: “ἀγκύλov: with ἅρμα only here (likely in reference to the curved edge of the chariot body), otherwise an epithet of bows (3x Il./Od. ἀγκύλα τόξα);”

Our interpretation: 1) This is the only instance of ἀγκύλov (meaning crooked) with ἅρμα (meaning weapon) and 2) if ἀγκύλov occurs with τόξov (meaning bow) at a frequency above the expected frequency, it is an epithet.

0) Requirements

To run and tweak this analysis, you will need the following:

  • this notebook, with its home directory

  • the pandas library

1) ἅρμα

First, let’s import some necessary libraries. This second one is our bespoke query library for the Greek lit corpus.

import pandas as pd

2) Epithet Frequency

If ἀγκύλov occurs with τόξov at an observed frequency above the expected frequency, it is an epithet. Our expected frequency formula looks a little like this:

expected frequency = ( (instances of ἀγκύλov with an object) * (instances of τόξov with an adjective) * 2 )/ (total word tokens in every work one of these words show up in)
